# LLMs for Model-driven Engineering: A Survey

This repository provides artifacts for the Systematic Literature Review (SLR) titled: _LLMs for Model-driven Engineering: A Survey_. 

During the SLR, we have the following raw data available:
* [`initial_search.csv`](data/initial_search.csv): This file contains the complete raw dataset obtained from the initial keyword search. It includes details of the 2,066 primary studies after removing duplicates across different databases.
* [`final_selection.csv`](data/final_selection.csv): This file presents the final set of 228 selected primary studies, derived from keyword searching and snowballing. The file covers raw data used to produce the results reported in the manuscript for the proposed research questions.
